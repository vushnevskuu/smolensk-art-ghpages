Как вернуть голосовое в ленту
---------------------------
1. Открой src/data/content-ru.json, найди массив "images".
2. Вставь содержимое docs/archive/voice-message-removed.json как новый элемент массива (с запятой между объектами, валидный JSON).
3. Перезапусти dev / обнови страницу.

Про файл .ogg
-------------
Запись была только по URL на Vercel Blob (см. поле url в архиве). Локальной копии в репозитории не было — из проекта удалены только данные ленты. Чтобы «удалить запись» с хостинга полностью, это нужно делать в панели Vercel / владельцем бакета.
